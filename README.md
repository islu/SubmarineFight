## Introduction
![screenshot](/screenshot.PNG)
**How To Play**
* `[->]`: Move ship go right
* `[<-]`: Move ship go left
* `[C]`: Drop the bomb
* `[R]`: Restart the game
* `[Esc]`: Exit the game
## Start!
* `Ruby2.6+`
* `gem install gosu`
* `ruby main.rb`
